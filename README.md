# Lizo Calc

Lizo Calc is a modern calculator platform built entirely with TypeScript, offering 45+ accurate calculators across multiple categories. The project is designed to provide fast, reliable, and easy-to-use calculation tools for students, professionals, and everyday users.

## Features

* 45+ calculators in one platform
* Built completely with TypeScript
* Clean and responsive user interface
* Fast calculation performance
* High accuracy results
* User-friendly design
* Continuously updated and improved
